<p align="center"><em>Hypixel Skyblock utility mod for Fabric 1.21.5</em></p>

## ⚙️ Features

### Galatea Island
- Foraging whisper tracker HUD (WIP)
- Tree progress overlay HUD
- Heart of the forest overlay (WIP)

### Mining
- Scatha Spawn Alerts
- Scatha Tracker HUD

### Misc
- Booster Cookie Reminder
- Missing God Potions Notifier
- Mouse Lock
- Timestamped chat messages
- Auto Copy RNG Messages
- SessionID Protection (WIP)

*...and many more to come!*

## 🗨️ Commands

- `/skyblock21` to open the config menu
- `/skyblock21 gui` to edit the position and scale of each HUD element. (`/s21 gui`)
- `/skyblock21 resettracker <tracker>` to reset trackers. (`/s21 resettracker`)
